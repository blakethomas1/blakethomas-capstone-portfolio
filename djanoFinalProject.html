Fake Twitter Application

Fake Twitter is a Python/Django-based web application mirroring the essential features of the popular social media platform, Twitter. This full-stack project demonstrates robust capabilities including user registration and authentication, post creation with image support, and a search functionality to explore posts. Designed to provide a simplified yet immersive microblogging experience, this application showcases my proficiency in Python, Django, and web development principles, emphasizing both back-end logic and front-end aesthetics
