Capstone Project

My capstone project is a MERN application that takes 2 sorting algorithms, compares the speed of them and outputs the results to the user. The project was a group effort of Blake Thomas, Trace Erickson, Hunter Ferrin, and Jaren Montano

User selecting algorithms

This video shows the user's ability to select two different algorithms to test their speed against each other.

It also shows the user's ability to join a queue.

User viewing results

This video shows the user's ability to see the results at the end of a run.

They are also able to see the resutls from thier prior runs and all other user's previous runs.

User viewing specifics about prior runs

This video shows the user's ability to see the specifics about a prior run.
